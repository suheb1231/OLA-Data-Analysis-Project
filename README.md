# OLA Data Analysis Project
The OLA Data Analysis Project is a SQL-based data analytics project focused on analyzing ride booking data to generate meaningful business insights. Using SQL Server, the project explores different aspects of Ola ride operations such as successful bookings, ride cancellations, customer behavior, payment methods, vehicle preferences, and driver ratings.

The project includes the creation of SQL queries and views to perform data analysis on the bookings dataset. Key analyses include identifying successful rides, cancelled rides by customers and drivers, UPI payment usage, Prime Sedan driver ratings, and booking trends.

# Tools & Technologies Used:
- SQL Server
- SQL Queries
- Views
- Excel Dataset
- GitHub

# Key Features:
- Booking status analysis
- Customer and driver cancellation analysis
- Payment method insights
- Vehicle type analysis
- Driver rating evaluation
- Aggregate functions and SQL views

This project demonstrates practical SQL skills including filtering, grouping, aggregation, sorting, and view creation for real-world business problem solving.
